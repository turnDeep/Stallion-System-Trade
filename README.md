# Stallion-System-Trade 🐎📈

**Stallion-System-Trade** は、米国株ラッセル3000銘柄を対象とした、完全自律型の「**損小利大・1日1回限定・オープニングレンジブレイクアウト（ORB）**」専用デイトレードBotシステムです。

Webull証券（米国株・現物取引）の公式API（V2 Trade / MDR Quotes）に完全対応し、週末の機械学習ベースの銘柄スクリーニングから、平日のリアルタイム超高速発注まで、Dockerコンテナ上で24時間365日**全自動**で稼働します。

実績として、過去60日間の未知のデータ（OOS）検証において、取引手数料（0.132%）を完全に加味した上で**約+19%の手数料負けしない驚異的な月利パフォーマンス**を叩き出しています。

## 🌟 システムの特徴（The "Stallion" Philosophy）

このシステムは「勝率や取引回数」を意図的に捨て、**「1回の爆発力（ボラティリティ）」**に全振りした尖ったハイリスク・ハイリターン哲学を持っています。

1. **AI（主成分分析）とボラティリティ・タームストラクチャーによる「最強の暴れ馬トップ10」の抽出:**
   毎週末、過去の膨大な5分足データからPCA（主成分分析）を実行。勝率ではなく「ADR（1日の平均値幅）」と「AvgWin（平均利益率）」が異常に高い、1日で10〜20%吹き上がるポテンシャルを持った「最強の種牡馬（Stallion）トップ10銘柄」を選別します。
   *※ADRの評価には「マルチタイムフレーム加重平均スコア（直近5日: 40%、15日: 30%、30日: 20%、60日: 10%）」を採用しており、過去の突発的なスパイク（ノイズ）を排除し、今まさに動意づいている旬のモメンタム銘柄だけを正確に捕捉します。*
2. **1日1回限定・資金90%の「一点集中フルインベストメント」:**
   取引回数が多いほど手数料負けするデイトレードの罠を回避するため、選抜されたトップ10銘柄の中で**「その日一番最初にブレイクアウトした1銘柄だけ」**に、総資金の90%を全ツッパ（フルインベストメント）します。
3. **完全無人・秒単位の執行（Webull MDRネイティブ対応）:**
   yfinance等の遅延APIを排除し、ミリ秒単位で戦うWebull公式のQuotes APIを直接叩くことで、9:35（EST）ジャストのオープニング高値を捕捉し、2秒間隔でブレイクアウトを監視します。

## 📦 アーキテクチャと稼働スケジュール

本システムは、OS標準のcronを使わず、Pythonの`schedule`ライブラリを使った安全な「マスター司令塔（Master Scheduler）」によって制御されています。サマータイム（夏時間）にもネイティブ対応しています。

* **`master_scheduler.py`**: 24時間常駐する司令塔スクリプト
  * **毎週日曜日 12:00 (EST):** `ml_pipeline_60d.py` を起動し、PCAスクリーニングを実施。翌週のターゲットとなるTop 10銘柄リストを更新します。
  * **毎週月〜金 09:25 (EST):** `webull_live_trader.py` を起動し、米国市場のオープンに備えます。
* **`webull_live_trader.py`**: デイトレード執行Bot
  * 09:30に市場が開き、最初の5分間（09:30〜09:35）の激しい攻防（Price Discovery）を見守ります。
  * **09:35** に、その5分間の「高値」をシステムに記録。
  * その直後から、その高値を上にブレイクアウト（ORB）した瞬間を検知し、WebullのV2 APIに「成行買い注文」を高速送信します。
  * **利確 (+10%)** または **損切り (-3%)** に到達するか、**15:55 (EST)** の大引け直前になった瞬間に全株を手放してその日のトレードを終了します（オーバーナイト・持ち越しリスク・ゼロ）。

## 🚀 構築手順（Deploy via Docker）

Xserver VPSやAWS EC2等のLinux環境で、以下のコマンドを実行するだけでシステムが永続稼働を開始します。

### 1. リポジトリのクローン
```bash
git clone https://github.com/turnDeep/Stallion-System-Trade.git
cd Stallion-System-Trade
```

### 2. 環境変数（API認証情報）のセット
プロジェクトのルートディレクトリに `.env` ファイルを作成し、APIキー等の情報を記述します。（`.env.example` を参考にしてください）
```env
WEBULL_APP_KEY="あなたのApp Key"
WEBULL_APP_SECRET="あなたのApp Secret"
WEBULL_ACCOUNT_ID="あなたの口座ID"
FMP_API_KEY="あなたのFMP API Key"
```

### 3. Docker Composeで一撃デプロイ
```bash
docker compose up -d
```
> ※ `Dockerfile` にてコンテナの内部時刻を米国（America/New_York）に強制指定しているため、サーバー側の日本時間を気にする必要はありません。
